       - Export `StatusCommand` extending `Command`
       - `static override paths = [["status"]]`
       - `json = Option.Boolean("--json", false, { description: "Output machine-readable JSON" })`
       - In execute(): call all three queries in parallel via `Promise.all([rpc("intent.query", {}), rpc("lock.query", {}), rpc("contract.query", {})])`
       - If --json: `this.context.stdout.write(JSON.stringify({ intents, locks, contracts }, null, 2) + "\n")` and return 0

       - Export `DeclareCommand` extending `Command`
       - `static override paths = [["declare"]]`
       - `description = Option.String("--description", { required: true, description: "Description of planned work" })`
       - `files = Option.Array("--files", { required: true, description: "Files this intent targets" })`
       - `symbols = Option.Array("--symbols", { required: false, description: "Symbol names within the files" })`
       - `json = Option.Boolean("--json", false, { description: "Output machine-readable JSON" })`
       - In execute(): read session ID via `getSessionId(WIT_DIR)`, call `rpc("intent.declare", { sessionId, description: this.description, files: this.files, symbols: this.symbols ?? [] })`
       - If --json: write full result JSON. Else: write "Intent declared: {intentId}" and if conflicts.hasConflicts, write each conflict item on a line with type and details.
       - Same try/catch error envelope pattern as StatusCommand.

    3. Create src/cli/commands/lock.ts:
       - Export `LockCommand` extending `Command`
       - `static override paths = [["lock"]]`
       - `symbol = Option.String("--symbol", { required: true, description: "Symbol path to lock (e.g. src/auth.ts:validate)" })`
       - `ttl = Option.String("--ttl", { required: false, description: "TTL in milliseconds" })`
       - `json = Option.Boolean("--json", false, { description: "Output machine-readable JSON" })`
       - In execute(): read session ID, call `rpc("lock.acquire", { symbolPath: this.symbol, sessionId, ttlMs: this.ttl ? parseInt(this.ttl, 10) : undefined })`
       - If --json: write result JSON. Else: write "Lock acquired: {symbolPath} (expires {expiresAt})" and list any warnings.
       - Same error envelope.
